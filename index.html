Git Command List

git init
git 저장소 초기화
디렉터리를 git으로 관리할 때 가장 먼저 할 일!
.git 디렉터리가 생성됨, 저장소 버전 관리하는데 필요한 파일이 들어있다
git status
git 상태 확인
git add [file, *]
파일 버전관리 스냅샷 생성
git commit -m '메모'
영구적으로 버전관리 스냅샷 파일 기록
git commit -a -m '메모 내용'
스테이지 상태 건너뛰고 커밋
git diff
스테이지 되지 않은 변경사항을 화면에 출력
git diff --staged
스테이지 이후 변경사항을 화면에 출력
git log
커밋 결과를 화면에 출력
