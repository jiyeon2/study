### 이미지 파일 확장자
- gif : 단색. 단순한 이미지 저장시. 용량 작음
- jpg : 사진 저장시. 압축률 높음
- png : 배경이 투명한 이미지 저장시
- svg : 벡터 이미지. 확대해도 깨지지 않음. 복잡한 도형, 그림은 용량 커지므로 단순한 이미지만 저장한다
